[![CI  finalization simulator](https://github.com/tgrospic/rchain-dev-finalization/actions/workflows/ci.yml/badge.svg)](https://github.com/tgrospic/rchain-dev-finalization/actions/workflows/ci.yml)

## Usage

### sbt console
```sh
# Enter sbt console 
sbt

# Compile
sbt> compile

# Compile with tests
sbt> Test/compile

# Run tests
sbt> test
```

### sbt CLI
```sh
# Compile
sbt compile

# Compile with tests
sbt "Test/compile"

# Run tests
sbt test
```
